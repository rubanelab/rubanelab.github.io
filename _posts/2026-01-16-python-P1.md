### Identifiers
- Contains only alphabets, digits and underscore (sholdn't start with digits)
- Case sensitive (car = "BMW", CAR = "Toyato" both are different variable)
- If starts with _ it's consider as private variable (_PI = 3.14159).
- If starts with __ it's consider as strongly private variable.
- If starts and ends with __, it will consider as language specific special identifiers. (\_\_init\_\_, \_\_del\_\_, \_\_main\_\_)
- Cannot use the keywords, following snippet will gives the list of keywords

```python
import keyword

keyword.kwlist

# 'False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await', 'break',

```

### Data types
Following are the fundamental data types.
- **int -** it can be represent in 4 ways (binary, octal, decimal, and hexadecimal)
- **float**
- **bool**
- **str**
- **complex**  x = a + bj (x.real, x.imag)

All fundamental data types are immutable.
